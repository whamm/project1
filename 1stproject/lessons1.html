HTML

General Bits of Information

  • Html = hyper text markup language
  • HTML has sideways triangles which are opening and closing in developer mode and in these you can see what is nested inside of the previous element
  • Html nests elements on the page inside of other elements like a Russian nesting doll. For example you have body and header tags nested inside of html tags and within body and header tags you have divs and p and so on nested.
  • HTML gives structure to the web
  • Everything on the web is a box
  • When working in HTML think from big to small in terms of containers

Tags

  • a for anchor tags and use href for the link
  • img for image tags and use the src attribute
  • br for line break
  • b for bold
  • em for italics
  • p for paragraph
  • div for dividing your page into boxes
  • span for applying css inline
  • More information about all tags can be found herehere

Elements

  • There are inline tags and block tags. Inline doesn't affect spacing and block tags have an invisible box around them
  • Elements have opening and closing tags(not all though) with content between

CSS

Boxes

  • Box structure goes from content>padding>border>margin
  • Border adnd padding are affected by the background color
  • Box Model Demo
  • Using display: flex can make it so divs(boxes) can be next to each other

Other CSS Info

  • CSS stands for Cascading Style Sheet
  • You can use CSS to assign styles to your HTML
  • Look for opportunities to apply styles and settings to larger chunks of html at first rather than repeat yourself in many different areas of a pages css. When you repeat too often you are likely to make mistakes
  • Other benefits of using css and using it effectively is it decreases the amount of code you have to write and the code that you write is more useful since it applies to more areas rather than just one.
  • CSS makes you page have a more unfied look
